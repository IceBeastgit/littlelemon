This project is a React-based web application for the Little Lemon restaurant that allows users to reserve a table online.

## Features
- Responsive layout using semantic HTML
- Booking form with date, time, guests, and occasion
- Form validation using HTML5 and React
- Dynamic available times based on selected date
- Confirmation page after successful booking
- Unit tests for reducers, form validation, and submission
- Accessibility improvements using labels and ARIA attributes

## Tech Stack
- React
- React Router
- CSS (Grid & Flexbox)
- Jest & React Testing Library
